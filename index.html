По необходимости могу писать на bootstrap 5 и materialize

При необходимости, код будет тщательно прокоментирован, со всеми необходимыми сносками.

По необходимости продемонстрирую как работает ваш сайт и как грамотно вносить изменения в его структуру не обращаясь за посторонней помощью. В том числе в письменном варианте.

Так же по необходимости могу писать на чистом HTML и CSS, Используя грид и флекс. Все также адаптивно и под любые устройства.

JavaScript

Обработка событий и обращение к DOM дереву, создание интерактивных элементов на сайте

Могу работать с bootstrap 5 и materialize, могу использовать их если предположим они уже подключены к вашему сайту, а также с не сложными элементами Vue.js

Готов разбираться в нестандартных для ситуациях, при необходимости исправлять ошибки

знаком с такими библиотеками как: list.js, sweetalert2.js, popper.js, baguetteBox.js, lightgallery.js, flatpickr.js, cleave.js, mixItUp3.js, swiper.js.

Прочее

Gulp - это инструмент для автоматизации рутинных задач, которые возникают при веб-разработке. В том числе минификация сайта или компиляния препроцессоров

К примеру данный сайт находится на бесплатном хостинге github-pages, кроме функции хостинка git предоставляет возможность командной разработки и доработки без внедрения в рабочий процесс
